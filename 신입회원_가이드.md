# 📚 스터디 신입회원 완벽 가이드

## 👋 환영합니다!

우리 스터디에 오신 것을 환영합니다!
이 문서는 스터디 운영 방식과 출석 시스템에 대해 **처음부터 끝까지** 안내합니다.
천천히 읽어보시고, 모르는 부분은 언제든 단톡방에 질문해주세요! 😊

## 🎯 스터디 소개

### 우리 스터디의 목표
```
✨ 매일 꾸준히 공부하는 습관 만들기
✨ 서로의 학습을 독려하고 응원하기
✨ 함께 성장하는 커뮤니티 만들기
```

### 핵심 원칙
- **자율성**: 무엇을 공부할지는 본인이 결정
- **꾸준함**: 매일 조금씩이라도 인증
- **존중**: 서로의 학습 방식과 속도 존중

## ⚙️ 신입회원 초기 설정 (처음 한 번만!)

### 1단계: 구글 드라이브 접근 권한 받기

#### 📧 운영진에게 전달할 정보
```
1. 본인 구글 계정 이메일 주소
2. 출석표에 표시될 이름 (닉네임 가능)
```

#### ✅ 초대 수락 방법
1. 이메일 확인 (구글 드라이브 공유 초대)
2. "열기" 또는 "수락" 버튼 클릭
3. 구글 드라이브에서 공유 폴더 확인

---

### 2단계: 본인 폴더 구조 파악하기

#### 📁 폴더 구조
```
공유 드라이브/
  └─ 스터디/
      └─ [본인 이름]/
          ├─ 2025-11-01/
          │   └─ 공부내용.md (또는 .txt, .pdf, 이미지 등)
          ├─ 2025-11-02/
          │   └─ 학습일지.md
          └─ 2025-11-03/
              ├─ 오늘공부.md
              └─ screenshot.png
```

#### 📝 날짜 폴더 형식

**올바른 예시:**
```
✅ 2025-11-26/
✅ 20251126/
✅ 2025_11_26/
```

**잘못된 예시:**
```
❌ 11월 26일/
❌ 오늘공부/
❌ 날짜폴더 없이 파일만 업로드
```

### 3단계: 출석표 북마크 저장

#### 🔖 출석표 URL
```
[출석표 URL을 여기에 입력]
```

1. 위 URL 접속
2. 브라우저 북마크에 저장 (Ctrl+D / Cmd+D)
3. 본인 이름이 출석표에 있는지 확인

### 4단계: 장기오프 신청 방법 숙지

#### 📋 장기오프란?
여행, 시험, 개인 사정 등으로 **3일 이상** 인증이 어려울 때 사용

#### 📝 신청 방법
```
1. 구글 폼 접속: [장기오프 신청 폼 URL]
2. 오프 기간 입력 (시작일 ~ 종료일)
3. 사유 간단히 작성
4. 제출!
```

#### ⚠️ 중요!
```
🚨 반드시 사전 신청! (사후 신청 불가)
🚨 최소 하루 전에 신청 권장
```

## 📋 출석 규칙 상세 안내

### ✅ 출석 인정 기준

#### 출석으로 인정되는 경우
```
✅ 해당 날짜 폴더에 파일이 1개 이상 있음
✅ 파일 형식은 자유 (md, txt, pdf, 이미지, 코드 등)
✅ 내용 길이 제한 없음 (한 줄이라도 OK!)
```

#### 출석으로 인정되지 않는 경우
```
❌ 날짜 폴더 없이 파일만 업로드
❌ 빈 파일 업로드
❌ 날짜 형식이 잘못된 폴더
```

---

### ⏰ 제출 마감 시간

```
📌 마감: 익일 새벽 3시까지

예시)
- 11월 26일 출석 → 11월 27일 새벽 3시까지 제출
- 밤 늦게 공부해도 새벽 3시 전에만 올리면 OK!
```

#### 시간대별 출석 처리
| 제출 시간 | 출석 날짜 |
|-----------|-----------|
| 11/26 오전 9시 | 11/26 출석 ✅ |
| 11/26 오후 11시 | 11/26 출석 ✅ |
| 11/27 새벽 2시 | 11/26 출석 ✅ |
| 11/27 새벽 4시 | 11/27 출석 ✅ (26일은 결석) |

---

### 📅 주간 인증 요구사항

#### 기본 규칙
```
📌 주 단위: 월요일 ~ 일요일 (7일)
📌 필요 인증: 주 4회
📌 오프 사용: 주 3회까지 (초과 시 결석 처리)
```

#### 예시로 이해하기
```
[좋은 예시 - 주 4회 인증]
월: ✅ 출석
화: ✅ 출석
수: 🏖️ 오프
목: ✅ 출석
금: 🏖️ 오프
토: ✅ 출석
일: 🏖️ 오프
→ 인증 4회 + 오프 3회 = 완벽! ✨

[나쁜 예시 - 오프 초과]
월: ✅ 출석
화: 🏖️ 오프
수: 🏖️ 오프
목: 🏖️ 오프
금: 🏖️ 오프 ← 4번째 오프! 결석 처리됨
토: ✅ 출석
일: ✅ 출석
→ 오프 3회 초과분은 결석으로 전환! ⚠️
```

### 🏝️ 오프 제도 상세

#### 오프 vs 장기오프
| 구분 | 오프 | 장기오프 |
|------|------|----------|
| 신청 | 불필요 (자동 처리) | 구글 폼 사전 신청 |
| 횟수 제한 | 주 3회 | 제한 없음 |
| 초과 시 | 결석 처리 | 해당 없음 |
| 사용 시기 | 단기 휴식 | 여행, 시험 등 장기 부재 |

#### 장기오프 기간 중 혜택
```
✨ 해당 기간 결석 카운트 안 됨
✨ 주간 필요 인증 횟수에서 차감
   예) 7일 중 3일 장기오프 → 필요 인증 4회 → 3회로 조정
```

---

### ⚠️ 경고 및 벌칙 제도

#### 단계별 안내
```
1~2회 결석: ✅ 정상
            → 아직 괜찮아요! 화이팅!

3회 결석:   ⚠️ 경고
            → 주의가 필요해요! 힘내세요!

4회+ 결석:  🚨 벌칙 대상
            → 벌칙 규정에 따라 처리됩니다
```

#### 결석 카운트 방식
```
📌 월 단위로 계산 (매월 1일 초기화)
📌 주간 집계 기준으로 결석 확정
📌 진행 중인 주는 결석 미확정 (일요일 지나야 확정)
```

## 🎯 일상 사용법 (매일)

### 매일 해야 할 일

```
1️⃣ 오늘 날짜 폴더 생성
   → 2025-11-26/

2️⃣ 공부 내용 파일 저장
   → 공부인증.md, 스크린샷.png 등

3️⃣ 끝!
   → 시스템이 자동으로 출석 체크
   → 출석표에서 확인 가능
```

### 💡 꿀팁

#### 1. 간단해도 OK!
```markdown
# 2025-11-26 공부 인증

오늘 배운 것: JavaScript 화살표 함수
```
→ 이 정도면 충분합니다!

#### 2. 이미지만 올려도 OK!
```
2025-11-26/
  └─ 공부인증.jpg
```
→ 스크린샷 한 장이라도 출석 인정!

#### 3. 주 초반에 4회 채우기
```
월화수목에 4회 인증 완료
→ 금토일은 마음 편하게 오프 가능!
```

## ❓ 자주 묻는 질문 (FAQ)

### Q1. 뭘 올려야 하나요? 형식이 정해져 있나요?
```
A: 아니요! 완전 자유입니다.
   - 텍스트 파일 (.md, .txt)
   - 이미지 (.jpg, .png)
   - PDF 문서
   - 코드 파일
   - 손글씨 사진
   뭐든 OK! 공부했다는 흔적만 있으면 됩니다.
```

### Q2. 하루에 여러 파일 올려도 되나요?
```
A: 네! 한 날짜 폴더에 여러 파일 가능합니다.
   2025-11-26/
     ├─ 오전공부.md
     ├─ 오후공부.md
     └─ 스크린샷.png
```

### Q3. 새벽에 공부하면 어느 날짜로 올려야 하나요?
```
A: 새벽 3시 기준으로 판단하세요!

   11/26 밤 11시 ~ 11/27 새벽 3시 사이 공부
   → 11/26 폴더에 올리면 11/26 출석

   11/27 새벽 3시 이후 공부
   → 11/27 폴더에 올려야 11/27 출석
```

### Q4. 깜빡하고 인증을 못 했어요. 소급 적용 되나요?
```
A: 안타깝지만 안 됩니다 😢
   - 마감(새벽 3시) 이후 소급 불가
   - 다음부터 알람 설정 추천!
```

### Q5. 장기오프 사후 신청 가능한가요?
```
A: 불가합니다!
   - 반드시 사전 신청 필요
   - 급한 일이 생기면 운영진에게 먼저 연락
```

### Q6. 출석이 안 올라갔어요!
```
A: 체크리스트 확인해보세요:

   [ ] 날짜 폴더 형식이 맞나요? (2025-11-26)
   [ ] 폴더 안에 파일이 있나요?
   [ ] 파일이 빈 파일은 아닌가요?
   [ ] 공유 폴더에 제대로 들어갔나요?

   다 확인했는데도 안 되면 단톡방에 문의!
```

### Q7. 결석 3회인데 벌칙 받나요?
```
A: 아니요! 3회는 "경고" 단계입니다.
   - 3회: ⚠️ 경고 (벌칙 아님, 주의 단계)
   - 4회+: 🚨 벌칙 대상
```

### Q8. 이번 달 결석이 다음 달로 이어지나요?
```
A: 아니요! 매월 1일에 초기화됩니다.
   - 11월 결석 3회 → 12월 1일부터 0회로 시작
```
