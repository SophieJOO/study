## ⚙️ 신입회원 초기 설정 (처음 한 번만! 약 10분 소요)

앞으로는 **평소처럼 Obsidian에서 공부 인증만 하면** 자동으로 출석이 체크됩니다!
딱 한 번만 아래 설정을 따라해주시면 됩니다.

---

### 1단계: Google Drive Desktop 앱 설치

#### 🪟 Windows 사용자
```
1. https://www.google.com/drive/download/ 접속
2. "Windows용 다운로드" 버튼 클릭
3. 다운로드한 GoogleDriveSetup.exe 실행
4. 설치 완료 후 Google 계정 로그인
```

#### 🍎 Mac 사용자
```
1. https://www.google.com/drive/download/ 접속
2. "Mac용 다운로드" 버튼 클릭
3. 다운로드한 .dmg 파일 실행
4. Google Drive 앱을 Applications 폴더로 드래그
5. 앱 실행 후 Google 계정 로그인
```

#### ✅ 설치 확인
```
- Windows: 작업 표시줄에 Google Drive 아이콘 나타남
- Mac: 메뉴바에 Google Drive 아이콘 나타남
→ 아이콘이 보이면 설치 성공! 🎉
```

### 2단계: Obsidian 폴더 동기화 설정

#### 📁 권장 폴더 구조
```
Obsidian 볼트/
  └─ 스터디인증/
      ├─ 2025-11-01/
      │   ├─ 공부인증.md
      │   └─ screenshot.png
      ├─ 2025-11-02/
      │   └─ 학습일지.md
      └─ 2025-11-03/
          ├─ 오늘공부.md
          └─ image.jpg
```

#### 📝 동기화 설정 방법

**방법 A: Obsidian 볼트 전체 동기화**
```
1. Google Drive Desktop 아이콘 클릭
2. ⚙️ (설정) → "환경설정" 클릭
3. "내 노트북" 탭 선택
4. "폴더 추가" 버튼 클릭
5. Obsidian 볼트 폴더 선택
6. "동기화" 선택 → "완료"
```

**방법 B: 스터디인증 폴더만 동기화 (추천! 💡)**
```
1. Obsidian 볼트 내에 "스터디인증" 폴더 생성
2. Google Drive Desktop 설정에서 해당 폴더만 동기화
→ 용량 절약 + 빠른 동기화!
```

#### 🧪 동기화 테스트
```
1. Obsidian "스터디인증" 폴더에 테스트 파일 생성
2. 웹에서 Google Drive (drive.google.com) 열기
3. 1~2초 후 파일이 나타나면 성공! ✅
```

## 📋 출석 규칙 상세 안내

### ✅ 출석 인정 기준

#### 출석으로 인정되는 경우
```
✅ 해당 날짜 폴더에 파일이 1개 이상 있음
✅ 파일 형식은 자유 (md, txt, pdf, 이미지, 코드 등)
✅ 내용 길이 제한 없음 (한 줄이라도 OK!)
```

#### 출석으로 인정되지 않는 경우
```
❌ 날짜 폴더 없이 파일만 업로드
❌ 빈 파일 업로드
❌ 날짜 형식이 잘못된 폴더
```

#### 📝 날짜 폴더 형식

**올바른 예시:**
```
✅ 2025-11-26/
✅ 20251126/
✅ 2025_11_26/
```

**잘못된 예시:**
```
❌ 11월 26일/
❌ 오늘공부/
❌ 날짜폴더 없이 파일만 업로드
```

### 🏝️ 오프 제도 상세

#### 오프 vs 장기오프
| 구분 | 오프 | 장기오프 |
|------|------|----------|
| 신청 | 불필요 (자동 처리) | 구글 폼 사전 신청 |
| 횟수 제한 | 주 3회 | 제한 없음 |
| 초과 시 | 결석 처리 | 해당 없음 |
| 사용 시기 | 단기 휴식 | 여행, 시험 등 장기 부재 |

#### 장기오프 기간 중 혜택
```
✨ 해당 기간 결석 카운트 안 됨
✨ 주간 필요 인증 횟수에서 차감
   예) 7일 중 3일 장기오프 → 필요 인증 4회 → 3회로 조정
```

2025-11-26/
  └─ 공부인증.jpg

## ❓ 자주 묻는 질문 (FAQ)

### Q1. 뭘 올려야 하나요? 형식이 정해져 있나요?
```
A: 아니요! 완전 자유입니다.
   - 텍스트 파일 (.md, .txt)
   - 이미지 (.jpg, .png)
   - PDF 문서
   - 코드 파일
   - 손글씨 사진
   뭐든 OK! 공부했다는 흔적만 있으면 됩니다.
```

### Q2. 하루에 여러 파일 올려도 되나요?
```
A: 네! 한 날짜 폴더에 여러 파일 가능합니다.
   2025-11-26/
     ├─ 오전공부.md
     ├─ 오후공부.md
     └─ 스크린샷.png
```

### Q3. 새벽에 공부하면 어느 날짜로 올려야 하나요?
```
A: 새벽 3시 기준으로 판단하세요!

   11/26 밤 11시 ~ 11/27 새벽 3시 사이 공부
   → 11/26 폴더에 올리면 11/26 출석

   11/27 새벽 3시 이후 공부
   → 11/27 폴더에 올려야 11/27 출석
```

### Q4. 깜빡하고 인증을 못 했어요. 소급 적용 되나요?
```
A: 안타깝지만 안 됩니다 😢
   - 마감(새벽 3시) 이후 소급 불가
   - 다음부터 알람 설정 추천!
```

### Q5. 장기오프 사후 신청 가능한가요?
```
A: 불가합니다!
   - 반드시 사전 신청 필요
   - 급한 일이 생기면 운영진에게 먼저 연락
```

### Q6. 출석이 안 올라갔어요!
```
A: 체크리스트 확인해보세요:

   [ ] Google Drive Desktop 실행 중인가요?
       → 작업표시줄/메뉴바에 아이콘 확인

   [ ] 동기화 완료되었나요?
       → 웹 Google Drive에서 파일 확인

   [ ] 날짜 폴더 형식이 맞나요? (2025-11-26)

   [ ] 폴더 안에 파일이 있나요?

   [ ] 파일이 빈 파일은 아닌가요?

   [ ] 폴더 공유가 제대로 되어 있나요?
       → 방장이 볼 수 있는지 확인

   다 확인했는데도 안 되면 단톡방에 문의!
```

### Q7. 결석 3회인데 벌칙 받나요?
```
A: 아니요! 3회는 "경고" 단계입니다.
   - 3회: ⚠️ 경고 (벌칙 아님, 주의 단계)
   - 4회+: 🚨 벌칙 대상
```

### Q8. 이번 달 결석이 다음 달로 이어지나요?
```
A: 아니요! 매월 1일에 초기화됩니다.
   - 11월 결석 3회 → 12월 1일부터 0회로 시작
```

### Q9. Obsidian 말고 다른 앱으로 작성해도 되나요?
```
A: 네! Google Drive에 파일만 들어가면 됩니다.
   - Notion → 내보내기 → Drive 업로드
   - VSCode → 파일 저장 → Drive 동기화
   - 메모장 → 저장 → Drive 동기화
   - 손글씨 → 사진 찍어서 업로드
```

### Q10. 폴더명을 영어로 해도 되나요?
```
A: 네! "StudyProof", "DailyStudy" 등 자유롭게 가능합니다.
   단, 날짜 폴더는 꼭 YYYY-MM-DD 형식으로!
```
