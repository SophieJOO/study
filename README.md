# 📚 스터디 자동화 & 지식 공유 시스템

> 출석체크 자동화는 유지하면서, 카카오톡에서 실시간 지식 교류를 활성화하는 시스템

---

## 🎯 프로젝트 개요

### 문제 상황
- 기존: 카카오톡에서 매일 공부 내용 인증 → 활발한 지식 교류
- 자동화 이후: Google Drive 업로드만으로 출석 체크 → **지식 교류 사라짐**
- 요구사항: 자동화 유지 + 카톡 교류 활성화

### 해결 방법
**"스터디 일일 다이제스트" 시스템**
- 매일 자동으로 조원들의 공부 내용 수집 & 요약
- 예쁜 웹페이지로 시각화
- 원클릭으로 카카오톡 공유

---

## ✨ 주요 기능

### 1️⃣ 자동 콘텐츠 수집 & 가공
- 📝 **마크다운**: 제목 추출, 내용 요약
- 📕 **PDF**: 파일 링크 제공
- 🖼️ **이미지**: 썸네일 생성 (+ 선택적 OCR)
- 🤖 **자동 실행**: 매일 저녁 8시 자동 수집

### 2️⃣ 예쁜 웹페이지 생성
- 🎨 **카드형 레이아웃**: 조원별 공부 내용 시각화
- 📊 **통계 대시보드**: 출석/오프/결석 현황
- 📱 **모바일 최적화**: 스마트폰에서도 편하게
- 🔗 **파일 링크**: 클릭 한 번으로 원본 확인

### 3️⃣ 카카오톡 간편 공유
- 💬 **원클릭 복사**: "카톡 공유" 버튼 한 번에 클립보드 복사
- 📋 **개인별 공유**: 내 공부만 공유 가능
- 📢 **전체 공유**: 하루 전체 요약본 공유
- ⏱️ **1일 1분**: 방장만 링크 공유하면 끝

## 🚀 빠른 시작

### 소요 시간: 약 15분

1. **Apps Script 코드 추가** (5분)
   - `digest-functions.gs` 내용을 기존 코드에 추가
   - `digest-page.html` 파일 생성

2. **웹 앱 배포** (3분)
   - Apps Script에서 웹 앱으로 배포
   - URL 복사

3. **트리거 설정** (3분)
   - 매일 저녁 8시 자동 실행 설정

4. **테스트** (2분)
   - 수동 실행으로 테스트
   - 웹페이지 확인

5. **카톡 공유 시작** (2분)
   - 다이제스트 페이지에서 "전체 내용 카톡 공유" 클릭
   - 카톡에 붙여넣기

**👉 자세한 내용은 [INSTALLATION_GUIDE.md](./INSTALLATION_GUIDE.md) 참고**

## ❓ FAQ

### Q1. 기존 출석체크 시스템에 영향이 있나요?
**A**: 전혀 없습니다! 기존 시스템은 그대로 유지되고, 다이제스트 기능만 추가됩니다.

### Q2. 조원들이 추가로 해야 할 일이 있나요?
**A**: 없습니다! 기존처럼 Google Drive에 파일만 업로드하면 됩니다. 공유는 방장/담당자가 합니다.

### Q3. 카카오톡에 자동으로 메시지가 전송되나요?
**A**: 아니요. 카카오톡 API 제약으로 자동 전송은 불가능합니다. 하지만 원클릭 복사 버튼으로 1분이면 충분합니다.

### Q4. PDF나 이미지 내용도 요약되나요?
**A**: 기본 기능은 파일명과 링크만 제공합니다. 고급 기능(OCR, AI 요약)을 추가하면 가능합니다.

### Q5. 비용이 발생하나요?
**A**: 기본 기능은 무료입니다. 고급 기능(AI 요약, 대량 OCR)을 사용하면 API 비용이 발생할 수 있습니다.

### Q6. 다른 플랫폼(텔레그램, 슬랙)으로 변경 가능한가요?
**A**: 네! 텔레그램 봇 API, 슬랙 Webhook 등으로 자동 전송이 가능합니다. (가이드 별도 문의)
